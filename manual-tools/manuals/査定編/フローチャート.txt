┌───────────────────────────────────────────────────────────┐
│【不慮の事故の定義】                                                        │
├───────────────────────────────────────────────────────────┤
│                                                                            │
│ 以下の2条件を全て満たすこと                                               │
│                                                                            │
│ ①急激かつ偶発的な外来の事故であること                                    │
│   ┌─────────────────────────────────────────────┐             │
│   │【急激性】原因→結果が急迫している                               │             │
│   │  ×慢性・反復性・持続性の強いもの（職業病、公害、薬害等）       │             │
│   ├─────────────────────────────────────────────┤             │
│   │【偶発性】被保険者の意思に基づかず、予期し得ない原因             │             │
│   │  ×故意、重大な過失によるもの                                   │             │
│   ├─────────────────────────────────────────────┤             │
│   │【外来性】外的原因によること                                     │             │
│   │  ×内的原因（疾病または体質的要因）によるもの                   │             │
│   └─────────────────────────────────────────────┘             │
│                                                                            │
│ ②約款別表（ICD-10分類）のいずれかの分類項目に該当すること              │
│                                                                            │
│ 【不慮の事故に該当しない例】                                              │
│  ・日常動作（歩く、かがむ、起き上がる等）による受傷                      │
│  ・潜函病（潜水病）、高山病、日射病、熱中症（回避不能除く）              │

│  ①手術を施行した入院の退院日翌日から再入院日までが31日以内

│  ・診療報酬上で手術算定となるもの
│  ・放射線照射、温熱療法など約款別表で手術扱いとしているもの
│  ・手術中の麻酔、造影等の検査、薬剤投与など一連の処置
│  ・手術と同等の操作を伴う検査または処置
│
│  ※例外
│  ・災害の場合：術後疾患は第一次原因（災害）の治療として扱う
│  ・がんの場合：術後疾患は「がんの治療」として通常の通算判断

│ 【「同時」の定義】
│  ・手術室に入室してから退室するまでの一連性

│  ①一旦手術室を退室後に他の手術を施行した証明あり
│  ②計画された一連の手術（補完的・補助的手術）ではない
│
├─【同時】
│   │
│   └─ Q5-3：新生物根治放射線照射との同日施行か？

│             │
│             └─ NO
